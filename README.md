## Installation

### Prerequisites

```
npm or yarn
```

### Installing

Initialize package manager

```
yarn
```

Build bundle

```
yarn build
```
### Development

Initialize package manager

```
yarn
```

Run webpack dev server

```
yarn dev-server
```
